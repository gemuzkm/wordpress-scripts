Суть метода:

Мы подменяем вывод плагина на статичную картинку (обложку мануала) с кнопкой «Открыть».

Реальный код плагина (и сам PDF-файл) прячем внутри тега <template>. Браузер не выполняет код и не грузит файлы внутри этого тега, пока мы его не попросим.

При клике мы достаем код из <template>, вставляем на страницу и запускаем.

Вот готовое решение. Добавьте этот код в файл functions.php вашей темы или через плагин Code Snippets.

Этот код перехватывает шорткод [pdf-embedder] и оборачивает его в нашу логику.


Как проверить, что это работает:
Вставьте код в functions.php.

Откройте любую страницу с мануалом в режиме «Инкогнито».

Нажмите F12 -> Вкладка Network.

Обновите страницу. Вы увидите, что огромный PDF-файл не загружается. Загружается только картинка-обложка.

Нажмите кнопку «Читать мануал». Только тогда в Network пойдет загрузка самого .pdf файла и скриптов pdf.js.
